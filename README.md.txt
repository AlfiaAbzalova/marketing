## Прогнозирование покупок для интернет-магазина
Описание проекта
В этом проекте была создана модель для предсказания покупок клиентов интернет-магазина в ближайшие 90 дней. Целью является повышение эффективности маркетинговых активностей и увеличение продаж.

Данные
Использованы датасеты, включающие информацию о покупках (apparel-purchase) и сообщениях от магазина (apparel-messages). Целевой признак — совершит ли клиент покупку в течение 90 дней (классификация 0/1).

Основные этапы
Предобработка данных: удаление выбросов, проверка и обработка дубликатов (дубликаты в данных о покупках были интерпретированы как особенности записи и сохранены, а в данных о сообщениях удалены).
Создание признаков: общее количество покупок, общее количество товаров, средняя сумма покупки, самый популярный кластер, количество дней с последней покупки, сезонность клиента и др.
Перебор гиперпараметров с помощью GridSearch. Наилучшей моделью оказалась CatBoostClassifier.
С использованием лучшей модели на тестовой выборке была достигнута метрика ROC-AUC: 0.7106.
Результаты
Наиболее значимыми признаками оказались:

Общее количество покупок
Средняя сумма затрат на покупку (average spent)
Самый популярный кластер (most popular cluster)
Общее количество товаров (total items)
Общее количество покупок (total purchases)
